# Artineer 리뉴얼 프로젝트
- IDEA : IntelliJ Ultimate
- FrontEnd : HTML, CSS, JavaScript, jQuery
- Template Engine : Thymeleaf
- BackEnd : Spring Boot 2.7.2
- Language : Java 11
- Build : Gradle
- Packaging : Jar
- DB : MySQL
- ORM : JPA
- DevOps : Jenkins, AWS, Docker
